Who We Are

We build hardware.

PCB design, layout, bringup, firmware and testing. We have experience with everything from single prototypes to full product development and launch.

Imagine

At the center of anything great is an idea, the initial spark that motivates development. Idea brainstorming and refinement is the foundation of products. We can help with your initial concept and outline a path to success.

Define

Project definition is the most important step of the design cycle. It collaborates engineers, designers, researchers and market experts. Together determine the product requirements, timelines and budgets. Project definition is typically the most misunderstood and poorly executed on step of the cycle.

Design

This step varies greatly depending on the project definition. Is product to be consumed by ten people? Or ten thousand? Or ten million? Who is going to consume it? Where do these people live? What technical capacity do they have? These are all question that greatly impact the system architecture and influence the design.

Prototype

Build, test, build, test. Test under different temperatures, vibrations, humidities, power sources, EMC, EMI, days of the week, device states, battery voltages, silicon corners, clock speeds, water ingress, component tolerance extremes and whatever else is relevant.

Scale

Once the design has been stress tested it's time to scale. The trick for hardware at scale is to not change anything unless absolutely required. This prevents the requirement to redo all the costly and time consuming regression testing. Hardware design at scale is a totally new game, you have to organize manufactures, supply chains and automated test equipment.

Launch and Maintain

Following the deployment you will most likely need to provide firmware updates for the device. You need to setup a continuous integration system that runs build verification tests (BVT). Firmware drops may require a certain level of regression testing to ensure nothing has broken. You need a platform to get the firmware to the end user, this may be an app, or a website. You need a bootloader so the user can easily flash the device. All these things need to be taken into account in step two.

What We Do

We are open.

We suggest using open source tools and libraries. Doing so enables fast and affordable development. Open source gives us the flexibility to modify and automate the tooling to our needs. When we're done with the design, you won't need to shell out tens of thousands of dollars to open the design files or compile the code.

Realtime Firmware

We develop hard realtime embedded firmware for arm microcontrollers. We have experience with all the modern realtime operating systems, such as FreeRTOS, Zephyr and ChibiOS. We develop with crossplatform open source tools such as gcc, make and gdb.

Circuit Design

We design, simulate, layout and test PCBs in house. We have experience with Altium, however we prefer to use Kicad if the design permits. With KiCad it becomes possible to the client to modify the design without having to purchase expensive software licenses. We have developed several Python frameworks around KiCad to augment it's capabilities.

Embedded Linux

Embedded Linux is extremely powerful. Once you have a working image up an running tasks that require networking, dynamic memory, security, filesystems, databases, language support, video or LCD screen become trivial. Leveraging tools like Buildroot and Yocto we can design extremely powerful systems for use in IOT, machine learning, machine vision or whatever your product needs may be.

Robotics

We have experience in designing and building robots. Software like Robot Operating System (ROS) are extremely powerful tools for implementing algorithms for control of robots. We have implemented several SLAM algorithms, such as LSD SLAM and ORB SLAM for localizing robots. We have experience with the latest Nvidia embedded systems for robotics and machine learning.

Featured Works

Here is some of our work.

FSR

Embedded Systems

Project Link

This is an embedded system for a client that need a Force Sensitive Resistor (FSR) array on a flex PCB to be used with a robotic gripper. It boasts an STM32F1 running freeRTOS and libopencm3. It samples data from the sensors and acting as a USB serial device, send the data back to the host ROS PC using rosserial.

Hydroponic Controller

Embedded Systems / IOT

Project Link

This is an embedded PCB designed to control a hydroponics system. It boasts an STM32F4 and esp8266 wifi controller. It has three fets, various switching power management ICs (PMICs) and a stepper motor driver. It uses FreeRTOS and libopencm3.

SoundHive

Analog Design

Project Link

Soundhive is a product designed to reduce e-waste. Thousands of high quality speakers end up in the landfill because their driving circuitry is dead. Soundhive provides a modular approach to bringing old speakers back to life. The modules stack vertically to allow for 1-8 speakers in a system. There is a battery backpack that enables the system to be portable. The PCB itself boasts a TI 40W class D high efficiency amplifier.

BirdEar

Embedded ML

Project Link

BirdEar is an embedded ML platform designed to classify species of birds using acoustic data in the wild. It uses tensorflow lite for microcontrollers to implement a neural net on an STM32F7. Is contains all the analog circuitry to filter the input and adjust the amplitude. It is extremely lower power and only wakes up when it hears a noise, since it's a bare metal system wakeup time is extremely fast and sleep power is extremely low. It logged data to an SD card using fatfs.

Page Up

Low Power Embedded Design

Project Link

Page Up is a wireless pager designed for calling servers at restaurants. It hosts an embedded design that lasts years on two AA cells. It uses a 2.4Ghz radio to send the data back to a linux SBC which then moved it to the cloud. The data was consumed by Android apps running on the servers phones. We did not design the app.

Logic Crusher

Embedded + Analog Design

Project Link

Logic Crusher is a Eurorack module used to sample and playback audio in a modulated form. Is uses a STM32 DSP core to implement filters and modulators on audio signals.
