Who We Are

We build hardware.

PCB design, layout, bringup, firmware and testing. We have experience with everything from one off prototypes to full product development and launch.

Immigine

At the center of anything great is an idea, the initial spark that fuels the following development. Idea brainstorming and refinement is the foundation of products. We can help with your initial concept and outline a path to success.

Define

Project definition is the most important step of the design cycle. This is when the engineers, designers, researchers and market experts sit down together and carve out the requirements. This is generally the most misunderstood and poorly executed on step of the cycle.

Design

Based on the project definition this step varies greatly. Is product to be consumed by ten people? Or ten thousand? Or ten million? Who is going to consume it? Where do these people live? These are all question that greatly impact the system architecture and influence the design.

Prototype

Time to build and test the thing. Based on the complexity of the design the boards may be populated in house or farmed out to a local fabrication shop. Test, test, test, under all conditions. Temperature, vibration, humidity, power sources, EMC, EMI, etc. Do all this before proceeding to a larger build.

Scale

Once the design has been stress tested it's time to scale. It is recommended to stay with the same contract manufacture barring any issues. Working with CM's is a trust building exercise, and the trust cuts both ways. Past a certain build quantity you need to develop automated test equipment (ATE). This is used to find build errors of the line.

What We Do

We are open.

We suggest using open source design tools, this enables a fast and more affordable transition. Open source gives us the flexibility to modify and automate the tooling to our needs. Also open source is awesome.

Realtime Firmware

We develop hard realtime embedded firmware for microcontrollers. We have experience with all the modern realtime operating systems, such as FreeRTOS, Zephyr and ChibiOS. We develop with crossplatform open source tools such as gcc, make and gdb.

Circuit Design

We design, simulate, layout and test PCBs in house. We have experience with Altium, however we prefer to use Kicad if the design permits. With KiCad it becomes possible to the client to modify the design without having to purchase expensive software licenses.

Embedded Linux

Embedded Linux is just getting more prolific. Leveraging tools like Buildroot and Yocto we can design extremely powerful systems for use in IOT, machine learning, machine vision or whatever task you need.

Robotics

We have experience in designing and building robots. Software like Robot Operating System (ROS) are extremely powerful tools for implementing algorithms for control of robots. We have implement several SLAM algorithms, such as LSD SLAM and ORB SLAM for localizing robots.

Featured Works

Here is of our work.

FSR

Embedded Systems

Project Link

This is an embedded system for a client that need a Force Sensitive Resistor (FSR) array for a robotic gripper. It boasts an STM32F1 running freeRTOS and libopencm3. It samples data from the sensors and acting as a USB device, send the data back to the host ROS PC using rosserial.

Hydroponic Controller

Embedded Systems / IOT

Project Link

This is an embedded PCB designed to control for a hydroponics system. It boasts an STM32 and esp8266 wifi controller. It has three fets and various switching PMICs.

SoundHive

Analog Design

Project Link

Soundhive is a product designed to reduce ewaste. Thousands of high quality speakers end up in the landfil because their driving circuitry is dead. Soundhive provides a modular appoach to bringing old speakers back to live. The modules stack vertically to allow for 1-8 speakers in a system. There is a battery backpack that enables the system to be portable. The PCB itself boasts a 40W class D high effiency amplifier.

BirdEar

Embedded ML

Project Link

BirdEar is an embedded ML platform designed to classify species of birds using acoustic data in the wild. It uses tensorflow lite for microcontrollers to implement a neural net on an STM32F7.

Page Up

Low Power Embedded Design

Project Link

Page Up is a wirelss pager designed for calling servers at restoraunts. It hosts and embedded design that lasts years on two AA cells. It uses a 2.4Ghz radio to send the data back to a linux SBC which then moved it to the cloud. The data was consumed by Android apps running on the servers phones.

Logic Crusher

Logic Crusher is a Eurorack module uses to sample and playback audio.
